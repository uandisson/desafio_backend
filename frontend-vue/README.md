# Frontend-Vue

## Requisitos

- Node 14
- Vue JS
- Docker (opcional) -> recomendado

## Executar sem Docker

Instalar dependências
    
    $ npm install

Executar

    $ npm run dev
    
## Executar com Docker

Iniciar e Debugar

    $ cd frontend-vue/
    $ sudo docker-compose up
